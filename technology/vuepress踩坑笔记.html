          # 1. 使用自定义指令时build后报错

          npm run build
Error: render function or template not defined in component: BlogList

          主要时由浏览器的 API 访问限制引起 文档

          如果你正在使用，或者需要展示一个对于 SSR 不怎么友好的组件（比如包含了自定义指令），你可以将它们包裹在内置的 

          <ClientOnly>
  <NonSSRFriendlyComponent/>
</ClientOnly>

          # 2. Can't resolve 'sass-loader'

          npm run dev
vuepress Can't resolve 'sass-loader'

          //需要安装依赖项
yarn add -D sass-loader node-sass

          # 3. Vue packages version mismatch:

          npm run build
Vue packages version mismatch:
- vue@2.5.21
- vue-server-renderer@2.5.22
This may cause things to work incorrectly. 
Make sure to use the same version for both.

          应该是依赖项版本不同

          //在package.json中强制制定版本
"devDependencies": {
"vue-server-renderer":"^2.5.21"
}
//或者 更新vue
npm install vue -g

          # 4.vue-loader报错

          npm run build
Error: [vue-loader] vue-template-compiler must be installed 
as a peer dependency, or a compatible compiler implementation
must be passed via options.

          应该是版本不对

          //package.json 将2.5.21 改为
"devDependencies": {
"vue-template-compiler": "^2.5.22",
}
